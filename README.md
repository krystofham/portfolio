# Portfolio – Kryštof Ham

Osobní portfolio webová stránka prezentující moje dovednosti, projekty a kontaktní informace.

Web slouží jako přehledná osobní prezentace.

## Funkce
- Představení osoby a přehled dovedností
- Přehled projektů s odkazy na živé ukázky a GitHub repozitáře
- Kontaktní sekce
- Dark mode
- Responzivní design pro počítače i mobilní zařízení

## Použité technologie
- HTML
- CSS
- JavaScript
- GitHub Pages

## Struktura projektu
- `index.html` – O mně
- `projects.html` – Projekty
- `contact.html` – Kontakty
